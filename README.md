Number Guessing Game (C++)

This is a simple Number Guessing Game created using C++ as part of my CODSOFT C++ Internship (Task 1).

The program generates a random number, and the user tries to guess it.

Features:

•Generates a random number

•Takes user input

•Tells whether the guess is too high or too low

•Continues until the correct number is guessed

Language Used:

•C++

How to Run:

•Compile the code:

g++ number_guessing_game.cpp -o game


•Run the program:

./game

•Concepts Used:

Variables

Loops

Conditional statements

Random numbers
